Actions

Select one or more vector paths, then measure. Already-measured lines are updated in place.

Point elevations (3D)

Heights are in cm at each path point (P1, P2, …). Start = height where the run arrives at that point; End = height where it leaves. If start and end match, the point is level (purple dot). If they differ, there is a vertical change at that point (green/red). Along each segment, the run length combines plan distance with the height change from the previous point’s End to the next point’s Start (3D length). Click Apply elevations after editing.

Dots mode: height and difference at each point. Turn off to use pill badges at points instead.

Badge mode: pills sit on vertices. Segment slopes still appear on the line when enabled above.

Per point: arrives (start) → leaves (end). Leave blank to clear a point.

How it works

Plan length
1. Set scale in Calibrate (or per frame).
2. Draw paths with the Pen tool (one vector per run).
3. Select line(s) and click Measure. A length label appears at the path midpoint and updates when the line moves or reshapes.
4. Drag a label to reposition it; the position is kept until the path geometry changes.
5. Lines inside a calibrated frame use that frame’s scale automatically.

3D length (optional)
6. Select a measured line and enter start/end heights per point above, then Apply elevations. The main label shows total 3D run length (plan + height changes).
7. Canvas markers: purple = level at a point, green/red = up/down at a point, blue = slope at the segment midpoint. Point markers are centered on each path vertex.

Calibrate scale

Tell the plugin how many pixels equal a real distance. Select a straight vector drawn over something you know (e.g. a 50 m wall), enter that length in cm, choose Global or a specific frame, then click Calculate & save. All measured lines in scope will use this scale until you change it.

Units

Displayed lengths pick a readable unit: under 1 m → cm, 1 m–999.99 m → m, 1 km and above → km. Values show two decimal places. Internal math always uses cm.
